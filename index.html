Your sided dice rolled


"Your 7 sided dice rolled X!"
"Your 30 sided dice rolled X!"
"Your 100 sided dice rolled X!"

fun main() {
   
    val myFirstDice = Dice()
    val diceRoll = myFirstDice.roll()
    println("Your ${myFirstDice.sides} sided dice rolled ${diceRoll}!")

    myFirstDice.sides = 30
    println("Your ${myFirstDice.sides} sided dice rolled ${myFirstDice.roll()}!")
    
    myFirstDice.sides = 100
    println("Your ${myFirstDice.sides} sided dice rolled ${myFirstDice.roll()}!")
}

class Dice {
    var sides = 7

    fun roll(): Int {
        val randomNumber = (1..sides).random()
        return randomNumber
    }
}

To make your life easier, you can enable auto-imports so that Android Studio automatically imports any classes that are needed by the Kotlin code.

In Android Studio, open the settings editor by going to File > Other Settings > Preferences for New Projects.

Android：Android adb error

In JDK 7, since the signature algorithm is changed, an 【INSTALL_PARSE_FAILED_NO_CERTIFICATE】 error occurs and apk can not be installed.Therefore we need to apply the following patch to tools / ant / build.xml of Android SDK.
Android SDK in Android SDK r19.Apache Ant 1.8.3.
You need to add [-sigalg MD5withRSA-digestalg SHA1] to the arsigner command option.Apk (Android OS 4 ... system) built without applying this,Failure [INSTALL_PARSE_FAILED_NO_CERTIFICATES] will be displayed.
